# Copilot Instructions

This repository includes an agent directive file that automated assistants must read and follow before taking actions.

- **Primary directive**: Always open and read the file at `.github/agents/test-journal-logger.agent.md` before performing any task in this repository.
- **Follow instructions**: Treat the directives in `.github/agents/test-journal-logger.agent.md` as authoritative for journaling, logging, and agent behavior (for example: updating `JOURNAL.md` after each prompt).
- **If unclear**: Ask for clarification rather than guessing or taking unilateral actions.

Assistants should consult the `.github/agents/test-journal-logger.agent.md` file at the start of every session and adhere to its instructions.
